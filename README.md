# TreasureHunter2

Production-ready C# Treasure Hunter AR application using Clean Architecture.

## Tech Stack
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- JWT Authentication
- .NET MAUI Blazor Hybrid
- MediatR CQRS Pattern

## Features
- GPS-based treasure hunting
- AR visualization
- Real-time proximity detection
- Gamification with points and inventory
- Hot/Cold sensor feedback
